E-Learning Platform – SQL Database Project

##  Project Overview
This project is focused on designing and implementing a **relational database for an E-Learning Platform** using SQL.  
It simulates a real-world online learning system where students enroll in courses, instructors manage content, and the platform tracks learning progress, payments, and reviews.

The project demonstrates strong fundamentals of **database design, normalization, relationships, and analytical SQL queries**.

---

##  Tools & Technologies
- SQL
- MySQL
  

---

## Database Features
- User management (Students & Instructors)
- Course and category management
- Student enrollments
- Lesson and progress tracking
- Payment records
- Course reviews and ratings

---

##  Database Tables
- `users`
- `courses`
- `categories`
- `enrollments`
- `lessons`
- `progress`
- `payments`
- `reviews`

Each table is designed using proper **primary keys, foreign keys, and constraints** to maintain data integrity.

---

##  SQL Concepts Used
- SELECT queries
- WHERE, GROUP BY, HAVING
- INNER JOIN & LEFT JOIN
- Subqueries
- Aggregate functions (COUNT, SUM, AVG)
- Window functions
- Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE)

---

##  Business Use Cases
- Identify the most popular courses
- Track student enrollment trends
- Monitor learner progress
- Analyze instructor performance
- Calculate total and course-wise revenue
- Understand user feedback through ratings and reviews
